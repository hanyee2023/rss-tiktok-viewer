# RSS 图文视频阅读器
一款纯前端实现的 RSS 阅读器，支持**图片、视频**两类订阅源，兼容标准 OPML 导入，本地数据备份/恢复，无需后端，开箱即用。

## ✨ 功能特点
- 分类浏览：独立切换 图片 / 视频 内容
- 订阅管理：新增、编辑、删除、批量标为已读
- 自动已读：上下滑动浏览时自动标记内容为已读
- 收藏功能：支持图文、视频收藏，随时回看
- 通用导入：支持主流阅读器导出的 **OPML** 格式订阅源
- 数据备份：JSON 全量导出/导入，防止数据丢失
- 视频优化：播放器控件常驻显示，支持普通视频 & m3u8 直播流
- 纯前端：数据存储在浏览器本地 `localStorage`，隐私安全

## 📖 使用教程
### 1. 添加订阅源
1. 点击顶部「新建」按钮
2. 填写订阅名称、RSS 链接
3. 选择类型：`图片类` / `视频类`
4. 点击保存，自动加载内容

### 2. OPML 批量导入
1. 点击顶部「导入/导出」
2. 粘贴 OPML 文本 或 上传 `.opml/.xml` 文件
3. 点击「导入OPML订阅源」，自动批量添加订阅

### 3. 数据备份与恢复
- **导出备份**：进入导入导出页面 → 点击「导出JSON备份」，内容自动复制到剪贴板，粘贴保存到文档即可。
- **导入恢复**：将备份文本粘贴到输入框 → 点击「导入JSON数据」。

### 4. 收藏与回看
- 浏览内容时，点击右侧 ⭐ 按钮完成收藏
- 点击顶部「收藏」可查看所有收藏内容，支持查看、删除

## 🗂 数据说明
所有订阅、已读记录、收藏数据均保存在**当前浏览器本地**：
- 更换浏览器/设备：请先导出备份，再在新设备导入
- 清除浏览器缓存会丢失数据，建议定期导出备份
