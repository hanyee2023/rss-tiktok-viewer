# RSS 极速阅读器
轻量级图片 + 视频 RSS 订阅阅读器，纯前端单页应用。

## 版本
**V1.2（最新稳定版）**

## 功能特性
✅ 视频自动播放  
✅ 原生视频播放器（进度条、暂停、音量、全屏）  
✅ 多图左右滑动浏览  
✅ 图片 / 视频分类自动汇总  
✅ 订阅管理（新增/删除/刷新/标为已读）  
✅ 收藏功能  
✅ 本地缓存，无需服务器  

## 更新日志（V1.2）
- 优化新建订阅弹窗按钮：保存/关闭左右分布
- 修复视频播放器控件隐藏问题
- 恢复原生进度条、音量、全屏按钮
- 优化界面布局与体验
- 提升滑动流畅度

## 使用方法
1. 打开 index.html
2. 点击「新建」添加 RSS 地址
3. 支持图片 RSS、视频 MP4 RSS
4. 自动分类展示

## 技术栈
- 纯 HTML + CSS + JavaScript
- 本地 localStorage 存储
- 无框架、无依赖
